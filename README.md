# upload here README

Upload HERE!

## Features

使用快捷键或者鼠标右键，上传本地图片然后将上传结果插入到光标位置。

## Extension Settings

使用该插件，首先需要配置图片上传接口地址：

`Code -> Preferences -> Settings`

该接口返回值需要满足如下格式：

```json
{
    "data": {
        "url": "https://xxxx.png"
    }
}
```
